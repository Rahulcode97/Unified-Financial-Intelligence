# Unified Financial Orchestration Suite

An independently engineered full-stack ecosystem designed to automate payment reconciliation, financial reporting, and revenue analysis.

### 🚀 Live Demo
https://rahulcode97.github.io/Unified-Financial-Intelligence/portal-account.html

### 🛠 The Problem Solved
* **Efficiency:** Reduced reporting cycles from 1 week to <1 hour.
* **Accuracy:** Automated reconciliation across PayPal, CCAvenue, and Bank Transfers, eliminating GST/Tax mismatches.
* **Security:** Integrated Passkey authentication and Role-Based Access Control (RBAC).

### 💻 Tech Stack
* **Frontend:** React, HTML, CSS (Tauri for Desktop)
* **Backend:** Node.js, PHP, Python (Data Analysis)
* **Cloud:** Google Cloud, MySQL, NoSQL

### 📊 Key Impact
* Managed transaction volumes of **63 Cr+**.
* Unified sales reporting for **5 major event portfolios**.
* Identified critical revenue leakage through deep-dive refund analysis.
