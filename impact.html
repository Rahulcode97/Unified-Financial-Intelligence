Technical Architecture (How)

Polyglot microservices: each layer uses tools matched to the problem—web UI, fast APIs, analytics, and durable storage on cloud infrastructure.

Layer Technology
Frontend / Desktop React for the web experience; Tauri for a lightweight, secure native shell where a desktop app is required.
Backend & APIs Node.js and PHP for robust API handling; Python for data analysis, reconciliation logic, and refund flows.
Database / Cloud Hybrid SQL (MySQL) and NoSQL, hosted on Google Cloud for scalable, low-latency reads against large transaction sets.
Security Passkey (biometric / hardware) authentication and end-to-end encryption for transaction logs and sensitive financial payloads.
